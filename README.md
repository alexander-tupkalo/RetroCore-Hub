# RETROCORE://HUB

Immersive retro-futuristic frontend ecosystem built with Vanilla JavaScript, HTML5 Canvas API, CSS3 and Web Audio API.

RETROCORE://HUB is designed as a cinematic sci-fi operating system that unifies multiple standalone retro-inspired projects into one atmospheric digital universe.

## ABOUT

RETROCORE://HUB is a modular retro-futuristic frontend ecosystem inspired by:
- VHS aesthetics
- CRT terminals
- synthwave culture
- analog sci-fi interfaces
- 80s/90s cyberpunk media

The project combines games, interactive media modules, procedural audio systems and cinematic UI architecture into one unified immersive experience.

Built entirely without React, Three.js or external UI frameworks.

---

## TECH STACK

- Vanilla JavaScript (ES6 Modules)

- CSS3
- Canvas API

- persistent operator sessions
- personalized greetings

- cinematic command-line interaction
- multilingual EN / UA support
- procedural typing simulation
- lore-driven signal responses
